# Data & Outputs
 
The `data/` and `outputs/` folders (containing the datasets, intermediate results, and generated outputs ) are hosted on Google Drive.
 
🔗 **Google Drive Link:** [Project Data & Outputs](https://drive.google.com/drive/folders/1nAVNOpDm9SG94o5oZ4kdFe3IIZjGd_9h?usp=drive_link)
 
## Folder Structure
 
| Folder | Description |
|--------|-------------|
| `config/` | Configuration files (weights, thresholds, parameters) |
| `data/` | Raw and processed audio data |
| `outputs/` | Generated results, plots, and prediction outputs |
| `src/` | Source code (rule-based modules) |
